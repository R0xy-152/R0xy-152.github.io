IELTS-Mate AI 面向雅思学习者的 AI 词汇、图片记忆与口语考官练习工具。

项目目标是把查词、兴趣推荐、多模态记忆和口语练习放进同一个学习工作流中。 当前静态页用于作品集展示；真实 AI 图片生成与口语对话需要后端部署后开放。

Product Design

从“查单词”扩展到“记住并使用单词”

个性化推荐

用户可以选择兴趣标签，首页每日单词按任意标签匹配生成，默认覆盖编程、科技和金融。

稳定查词链路

查词结果结合本地缓存、词典 API、本地 fallback、免费翻译服务和 Gemini，降低网络波动影响。

AI 图片记忆

后端支持 Gemini 图片模型，根据单词和语义生成插图，并用每 IP 每天一次限制控制公开 Demo 成本。

雅思口语考官

实验性口语模块模拟 IELTS examiner，支持双向对话、语音输入输出和评价反馈。

部署安全

API key 只保存在后端环境变量中，不放进前端或 GitHub；静态页面只展示项目，不调用敏感接口。

测试覆盖

后端核心流程使用 pytest 覆盖，包括每日单词、查词、图片额度、偏好设置和口语功能。

Implementation

轻量全栈原型，保留上线扩展空间

FastAPI SQLite / SQLAlchemy Vanilla JavaScript Gemini API pytest Render-ready config

今天这个页面可以直接作为静态作品集页面部署。后续如果接入 Render、Railway 或 VPS， 可以把这里的 GitHub 链接旁边再加一个 Live Demo 入口，让访客实际体验 AI 图片生成和口语对话。
